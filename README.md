# Engineering-Students-Placement-Dashboard
A Power BI dashboard project that analyzes engineering students’ placement data using interactive visualizations. The dashboard provides insights into placements, internships, certifications, projects, coding skills, salary trends, and branch-wise performance.

## Project Overview
This Power BI dashboard analyzes engineering students' placement data and provides insights into placements, internships, certifications, coding skills, projects, and salary trends.

## Tools Used
- Power BI
- SQL
- Python
- Excel

## Features
- Branch-wise placement analysis
- Coding skill analysis
- Internship and certification insights
- Average salary by branch
- Interactive filters and charts

## Key Insights
- CSE students have higher placement rates.
- Students with better coding skills get more placements.
- Internships and certifications improve placement chances.
